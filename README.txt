== Description ==
Professional apartment booking solution powered by WooCommerce. Offers a flexible system with calendar, icons, testimonials, and date selection.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ov-booking/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Booking Settings page to configure behavior (shortcode or override mode).

== Shortcode ==
Use `[ovb_booking_single]` to insert booking template manually.
